# Anomaly_Detection
## User Activity Anomaly Detection
### Overview

This project explores techniques to detect unusual or suspicious behavior in user activity logs using both statistical methods and machine learning approaches.

### Dataset

The dataset contains user activity logs including login behavior, data usage, file downloads, timestamps, and remote access indicators.

### Methodology

* Explored the dataset to understand activity patterns and feature distributions

* Visualized user activity using histograms and scatter plots

* Applied statistical anomaly detection techniques:
      1.Z-score
      2.Interquartile Range (IQR)

* Applied machine learning–based anomaly detection using Isolation Forest

* Compared anomalies detected by multiple methods to reduce false positives

* Identified suspicious user sessions based on multiple indicators

### Tools Used

Python, pandas, NumPy, scikit-learn, matplotlib

IQR outlier detection

Isolation Forest
